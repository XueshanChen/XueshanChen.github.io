Help

This is the help of tracker – An android app to track your outdoor activities.

You can contact me via email

index

The index page have the main functionalities of this app.

  • Start: user can start track their activities by click this button
  • Setting & Help menu, on the top left corner, user can visit these two buttons at any page
  • View History: user can view the recoded histories of their activities, which was stored in the database.

Tracker

This is the most important part of this app.

  • User can select one type of the activity to continue:

  • Setting parameters: user can set the priorities of the GPS/WIFI

  • By clicking Start button, the tracker start to record, user can also reset the priority

  • Finish, by click finish tracker, the tracker will stop, user can take a photo and select the weather of this activity, the relevant information will also be shown

Tracked Activities

  • View history: user can view all the tracked records

  • User can view the summary

  • User can share it to their friends:

  • User can view the detail and delete some items by click the top-left corner icon
