Agent-oriented programming (AOP) is much in demand in complex and dynamic environments. The Belief-Desire-Intention (BDI) system is a widely used framework for modelling the mental states of agents. This framework is used in many agent-oriented programming languages. GenGPT is a program that generates goal plan trees (GPTs) for modelling agent intentions as well as testing AOP languages, but its current propositional logic generation and lack of real-life examples limit its functionality. To address these limitations, we propose GenGPT-X, a program that uses first-order logic generation and real-life examples to improve the performance of BDI systems. The contribution of this paper is twofold. First, GenGPT-X, a program that extends the capabilities of GenGPT by exploiting first-order logic generation and real-life examples to improve the performance of the BDI system, is implemented. Second, a software engineering approach was used to develop the interface of GenGPT-X to improve the user experience. The results show that the use of first-order logic increases the adaptability of the GPT in different situations and that the user graphical interface to display the relationships of nodes in the GPT is more intuitive.
